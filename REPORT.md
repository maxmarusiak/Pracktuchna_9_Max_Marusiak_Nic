## 1. Класи для роботи з файловою системою в C#
Використовуються класи:

File — статичні методи для читання/запису файлів.

FileInfo — робота з конкретним файлом (розмір, дата, атрибути).

Directory — створення/видалення папок.

DirectoryInfo — детальна інформація про папку.

Path — робота з шляхами.
## 2. Різниця між File/FileInfo та Directory/DirectoryInfo
File / Directory — статичні класи, працюють без створення об’єкта.

FileInfo / DirectoryInfo — інстанси, дозволяють отримувати властивості (розмір, дата створення, атрибути) і виконувати операції над конкретним файлом/папкою.
## 3. StreamReader та StreamWriter
StreamReader — читає текстовий файл построчно.

StreamWriter — записує текст у файл.
Використовуються, коли потрібен контроль над потоком: буферизація, построчне читання, робота з великими файлами.
## 4. Серіалізація через System.Text.Json
Серіалізація працює через:

JsonSerializer.Serialize()

JsonSerializer.Deserialize()

Використані атрибути:

JsonPropertyName — задає ім’я поля в JSON.

JsonIgnore — виключає поле з серіалізації.
## 5. Винятки при роботі з файлами та їх обробка
Можуть виникати:

FileNotFoundException

IOException

UnauthorizedAccessException

JsonException

Обробка:

через try/catch

створено власний виняток InvalidFileFormatException

логування помилок у файл.
## 6. Реалізація резервного копіювання
Зроблено через клас BackupService:

копіювання файлу в папку Backups/

додавання timestamp до імені

очищення старих копій через CleanOldBackups(days).
